# Example Application

This is a simple example application demonstrating how to use tviewyaml.

## Running the Example

```bash
cd example
go run main.go
```

## Configuration Files

The example uses three configuration files in the `config/` directory:

- `root.yaml` - Defines the pages in the application
- `main.yaml` - The main menu page with a list
- `about.yaml` - An about page with a text view

## Navigation

- Use arrow keys to navigate
- Press the shortcut keys shown in the list (w, a, q)
- Press ESC to return to the main menu from any page
