# Example Application

A comprehensive demonstration of all tviewyaml features and widget types. This interactive example is the best way to learn how to use the package.

## Running the Example

```bash
cd example
go run main.go
```

## Widget Demonstrations

The example application includes working demonstrations of every supported widget:

| Widget | File | Key Features Demonstrated |
|--------|------|---------------------------|
| **Box** | `box.yaml` | Basic containers, borders, title alignment |
| **Button** | `button.yaml` | Interactive buttons with callbacks |
| **Checkbox** | `checkbox.yaml` | Checkboxes with state management |
| **Dropdown** | `dropdown.yaml` | Dropdown menus with options |
| **Flex** | `flex.yaml` | Horizontal/vertical layouts, proportions, fixed sizes |
| **Form** | `form.yaml` | Complete forms with all input types, validation |
| **Grid** | `grid.yaml` | Responsive grid layouts, spanning, borders |
| **InputField** | `inputfield.yaml` | Text input with validation, masks, acceptance functions |
| **List** | `list.yaml` | Selectable lists with shortcuts and callbacks |
| **Modal** | `modal.yaml` | Modal dialogs with custom buttons |
| **Table** | `table.yaml` | Data tables with headers, colors, selection |
| **TextView** | `textview.yaml` | Text display with colors, regions, alignment |
| **TreeView** | `treeview.yaml` | Hierarchical tree structures with expansion |

## Navigation & Controls

### Main Menu
- **Arrow keys** or **j/k**: Navigate through the list
- **Shortcut keys** (b, u, c, i, f, l, t, v, r, d, m, x, g): Jump directly to a demo
- **Enter**: Select an item
- **q**: Quit the application

### Within Demos
- **ESC**: Return to main menu from any page
- **Tab**: Move between form fields and focusable elements
- **Arrow keys**: Navigate tables, lists, and tree structures
- **Enter**: Select/activate items
